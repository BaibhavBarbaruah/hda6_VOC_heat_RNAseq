# hda6_VOC_heat_RNAseq


This repository contains the core data files and analysis script used to build the DESeq2 dataset for the RNA-seq experiment investigating VOC-primed heat stress responses in *Arabidopsis thaliana* (Col-0 and *hda6*).

### `data/`
- `gene_counts_matrix.txt`  
  Gene Ã— sample raw counts generated by featureCounts.
- `sample_table.csv`  
  Metadata containing sample names, genotype, and treatment.

### `scripts/`
- `01_build_dds_from_gene_counts_and_sample_table.R`  
  R script that:
  1. Reads the count matrix and metadata  
  2. Cleans sample names  
  3. Constructs the DESeq2 object (`dds`)  
  4. Runs `DESeq()`  
  5. Saves `dds_from_gene_counts_and_sample_table.RData`

### `results/`
- `dds_from_gene_counts_and_sample_table.RData`  
  Pre-built DESeq2 object ready for downstream differential expression analysis.
